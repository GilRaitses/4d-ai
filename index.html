Project Overview

This project develops a physics-informed machine learning system for **4D modeling** (3D spatial + time) of climate reconstruction. The Finger Lakes region serves as a case study, offering a well-preserved record of glacial history that helps refine computational models for simulating past environmental changes.

By integrating **remote sensing, geospatial modeling, and AI-driven simulation**, the system reconstructs terrain evolution, simulates ice flow and sediment transport, and provides an interactive tool for researchers to analyze glacial dynamics.

Methodology

The project integrates **AI-driven modeling with geophysical data** to reconstruct historical climate conditions and predict future environmental shifts.

  • Data Collection: Acquiring high-resolution DEMs, satellite imagery, and geological maps.
  • Physics-Informed ML: Using machine learning models that incorporate conservation laws to enhance simulation accuracy.
  • 4D Reconstruction: Developing time-dependent visualizations that model glacial formations and sediment transport.
