Project Overview

This project develops a physics-informed machine learning system for 4D modeling of glacial geomorphology. Focused on the Finger Lakes region—a natural laboratory with an extensive record of glacial processes—the system integrates high-resolution remote sensing data, geological maps, and simulation models to quantitatively reconstruct the evolution of glacial landforms.

The goal is to generate an interactive tool that not only documents past ice flow, sediment transport, and landscape evolution but also enables simulation of dynamic processes (such as subglacial floods and debris flows) to test new hypotheses about environmental change.

Methodology

My approach combines rigorous data analysis with physics-informed machine learning to build a comprehensive model of glacial evolution.

  • Data Acquisition: Obtain high-resolution DEMs, satellite imagery, and geological maps from sources like Google Earth Engine.
  • Model Development: Train physics-informed neural networks (PINNs) and deep operator methods to incorporate conservation laws (mass, momentum, energy) into the modeling process.
  • 4D Reconstruction: Build dynamic, time-dependent models that capture the evolution of glacial landforms—allowing interactive exploration of past and simulated processes.
Extended Methodology Details

The modeling pipeline starts with harmonizing various geospatial datasets into a unified reference system. Using existing simulation frameworks (e.g., FastScape, studies by Yuan et al., and GoSPL research), I initialize the ML models with realistic parameters derived from field data.

Physics-informed neural networks are then trained on this dataset, embedding physical laws into the learning process to predict processes like ice viscosity, sediment transport, and subglacial erosion. The resulting 4D model provides a cross-scale view of landscape evolution, offering both historical reconstruction and predictive simulation capabilities.

Domain-Specific Background

Understanding the evolution of glacial landforms requires a rigorous, quantitative approach. This project uses explicit physical models to derive geomorphic transport laws from field data.

Extended Domain Background

Detailed digital terrain analysis shows that the morphology of elliptical mounds (drumlins) reflects the interplay between ice dynamics, erosion, and sediment transport. By coupling high-resolution remote sensing data with numerical simulation techniques, key features such as orientation, slope, and curvature can be quantitatively analyzed to validate transport laws. This method helps in understanding how climate and ecological processes influence landscape evolution over time.
